有2个长度N的数组 A, B, 给出一个数子 m, m<N,
从 A 中选择 m 个数字,记录下标 k1,k2...km,  从 B 中选择剩余下标的数字. 使得 m 个数字 + N -m 个数字的和最大. 返回这个最大值


例如
```
m = 2
A = [4, 2, 1] , B = [2, 5, 3]

A 取 [4, 0, 1]   // 0 表示不选择
B 取 [0, 5, 0]

sum = 10
```
```
m = 2
A = [7, 1, 4, 4], B = [5, 3, 4, 3]

A: [7, 0, 0, 4]
B: [0, 3, 4, 0]

sum = 18
```
